## Overview

The Bargaining benchmark evaluates multi-agent collaboration on **negotiation and bargaining tasks**. It contains **100 tasks** based on real product listings, testing agents' ability to negotiate prices and terms while balancing competing interests between buyers and sellers.

## Task Characteristics

### Agent Configuration
- **Number of Agents**: 4 (1 buyer, 3 sellers or vice versa)
- **Agent Roles**: Buyers and sellers with competing interests
- **Collaboration Mode**: Hierarchical (parent-child relationships)
- **Relationship Type**: Parent relationships between lead agent and others

### Environment
- **Type**: Bargaining Environment
- **Max Iterations**: Variable (typically 10-15)
- **Tools**: Price offers, counteroffers, information sharing, negotiation termination
- **Products**: Real e-commerce items with actual prices and ratings

## Negotiation Tools

Agents have access to specific negotiation actions:

### Offer Price
```python
def offer_price(price: float, reason: str) -> dict:
    """
    Make a price offer with justification.
    
    Args:
        price: Proposed price in USD
        reason: Explanation for the price point
    
    Returns:
        Offer status and response
    """
    return {
        "action": "offer",
        "price": price,
        "reason": reason
    }

# Example
offer_price(45.00, "Market analysis shows similar products at $40-50")
```

### Reject and Counter
```python
def reject_and_counter(
    counter_price: float,
    reason: str,
    concessions: list = None
) -> dict:
    """
    Reject current offer and propose counter-offer.
    
    Args:
        counter_price: New proposed price
        reason: Explanation for rejection and counter
        concessions: Optional non-price benefits
    
    Returns:
        Counter-offer details
    """
    return {
        "action": "counter",
        "price": counter_price,
        "reason": reason,
        "concessions": concessions or []
    }

# Example
reject_and_counter(
    50.00,
    "Premium materials and 4.7-star rating justify higher price",
    concessions=["Free expedited shipping", "Extended warranty"]
)
```

### Provide Information
```python
def provide_information(info_type: str, details: dict) -> dict:
    """
    Share relevant information to support negotiation position.
    
    Args:
        info_type: Type of information (product, market, terms)
        details: Specific information to share
    
    Returns:
        Information package
    """
    return {
        "action": "inform",
        "type": info_type,
        "details": details
    }

# Example
provide_information(
    "product_quality",
    {
        "materials": "Premium steel construction",
        "weight_capacity": "50 lbs per tier",
        "warranty": "2 years"
    }
)
```

### End Negotiation
```python
def end_negotiation(outcome: str, final_terms: dict = None) -> dict:
    """
    Terminate negotiation with agreement or walk away.
    
    Args:
        outcome: "agree" or "walk_away"
        final_terms: Agreed terms if outcome is "agree"
    
    Returns:
        Negotiation conclusion
    """
    return {
        "action": "end",
        "outcome": outcome,
        "terms": final_terms
    }

# Example - Agreement
end_negotiation(
    "agree",
    {
        "price": 47.50,
        "shipping": "expedited (3 days)",
        "warranty": "standard 1 year"
    }
)

# Example - Walk away
end_negotiation("walk_away")
```

## Evaluation Metrics

### Deal Quality (35%)
- **Price Fairness**: How close to market value?
- **Mutual Benefit**: Do both parties gain?
- **Term Completeness**: Are all aspects covered?
- **Sustainability**: Is agreement viable long-term?

### Negotiation Process (30%)
- **Tool Usage**: Effective use of negotiation tools
- **Communication**: Clear and professional dialogue
- **Reasoning**: Logical justifications for positions
- **Flexibility**: Willingness to find middle ground

### Satisfaction (20%)
- **Buyer Satisfaction**: Needs met within budget
- **Seller Satisfaction**: Profitable and sustainable
- **Priority Alignment**: Key priorities addressed
- **Relationship Quality**: Potential for future deals

### Efficiency (15%)
- **Iteration Count**: Reaching agreement quickly
- **Concession Balance**: Fair give-and-take
- **Information Sharing**: Transparency where appropriate
- **Decision Quality**: Well-informed choices

## Running Bargaining Benchmarks

### Load and Execute Task

```python
import json
from marble import BargainingEnvironment, MultiAgentSystem

# Load bargaining task
with open('multiagentbench/bargaining/bargaining_main.jsonl') as f:
    tasks = [json.loads(line) for line in f]

task = tasks[0]

# Initialize bargaining environment
bargain_env = BargainingEnvironment(
    max_iterations=task['environment']['max_iterations']
)

# Create multi-agent system
mas = MultiAgentSystem(
    agents=task['agents'],
    environment=bargain_env,
    relationships=task['relationships']
)

# Run negotiation
result = mas.run(task['task']['content'])

print(f"Deal Reached: {result.agreement}")
if result.agreement:
    print(f"Final Price: ${result.terms['price']:.2f}")
    print(f"Iterations: {result.iterations}")
```

### Evaluate Negotiation

```python
from marble.benchmarks.bargaining import BargainingEvaluator

evaluator = BargainingEvaluator()

scores = evaluator.evaluate(
    negotiation_history=result.messages,
    final_terms=result.terms,
    product_info=task['task']['product'],
    buyer_priorities=task['task']['buyer_priorities'],
    seller_priorities=task['task']['seller_priorities']
)

print(f"Deal Quality: {scores.deal_quality:.2f}")
print(f"Process Quality: {scores.process:.2f}")
print(f"Buyer Satisfaction: {scores.buyer_satisfaction:.2f}")
print(f"Seller Satisfaction: {scores.seller_satisfaction:.2f}")
print(f"Overall Score: {scores.overall:.2f}")
```

### Batch Evaluation

```python
from marble.benchmarks import BenchmarkRunner

runner = BenchmarkRunner(

```

## Example Negotiation

### Task: Rolling Cart Negotiation

**Product**: 3-Tier Utility Cart ($46.99, 4.7â˜…)  
**Buyer Priority**: Timely delivery  
**Seller Priority**: Premium pricing justification

**Negotiation Flow**:

```python
# Round 1: Buyer opens
buyer.offer_price(
    40.00,
    "Similar 3-tier carts available at $35-45 range"
)

# Round 2: Seller responds
seller.reject_and_counter(
    46.99,
    "Premium steel construction and 4.7-star rating justify price",
    concessions=["Free standard shipping"]
)

# Round 3: Buyer emphasizes priority
buyer.provide_information(
    "delivery_requirement",
    {"needed_by": "5 days", "reason": "home organization project"}
)

# Round 4: Seller offers solution
seller.reject_and_counter(
    44.99,
    "$2 discount for volume, includes expedited shipping (3 days)",
    concessions=[
        "Expedited shipping (normally $8.99)",
        "Assembly video tutorial"
    ]
)

# Round 5: Buyer accepts
buyer.end_negotiation(
    "agree",
    {
        "price": 44.99,
        "shipping": "expedited (3 days)",
        "extras": ["assembly tutorial"],
        "warranty": "standard"
    }
)
```

**Outcome Analysis**:
- **Price**: $44.99 (buyer saved $2, seller maintains margin)
- **Delivery**: Expedited shipping meets buyer's timeline
- **Seller**: Added value without deep discount
- **Iterations**: 5 (efficient)
- **Satisfaction**: High for both parties

## Negotiation Strategies

### For Buyers

1. **Anchoring**: Start with reasonable but low offer
2. **Market Research**: Reference comparable products
3. **Priority Focus**: Emphasize key requirements
4. **Bundle Thinking**: Request package deals
5. **Walk-Away Willingness**: Have alternatives ready

### For Sellers

1. **Value Demonstration**: Highlight quality and features
2. **Concession Trading**: Offer non-price benefits
3. **Volume Opportunities**: Suggest bulk or future purchases
4. **Premium Justification**: Explain price differentiation
5. **Relationship Building**: Focus on long-term partnership

## Common Challenges

### Challenge 1: Price Gap
**Problem**: Large difference between buyer/seller expectations  
**Solution**: Find creative middle ground with concessions

### Challenge 2: Priority Mismatch
**Problem**: Buyer wants speed, seller wants commitment  
**Solution**: Package deal with expedited delivery + future order

### Challenge 3: Information Asymmetry
**Problem**: One party has more market knowledge  
**Solution**: Share relevant information transparently

### Challenge 4: Impasse
**Problem**: Neither party willing to move  
**Solution**: Introduce third-party mediation or walk away

## Best Practices

### For Effective Negotiation
1. **Start with research** on market prices and alternatives
2. **Listen actively** to understand other party's priorities
3. **Use tools appropriately** with clear reasoning
4. **Be flexible** on non-critical points
5. **Document agreements** clearly

### For Agent Design
1. **Model realistic behaviors** (not always adversarial)
2. **Include personality** traits (aggressive, cooperative, etc.)
3. **Implement learning** from previous negotiations
4. **Balance interests** appropriately
5. **Know when to walk away**
